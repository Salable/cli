# Create Plan

This command allows you to create a new plan on a product

## Code Sample

```bash
salable create plan
```

## Parameters

### --productName

_Type:_ `string`

The product to create the plan on

### --name

_Type:_ `string`

The name of the plan

### --displayName

_Type:_ `string`

The display name of the plan

### --description

_Type:_ `string`

The description of the plan

### --visibility

_Type:_ `"public" | "private"`

The visibility of the plan
