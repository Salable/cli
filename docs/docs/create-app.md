# Create App

This command allows you to create an example application consuming Salable

## Code Sample

```bash
salable create app
```

## Parameters

### --template

_Type:_ `string`

The type of template you'd like to generate.

### --name

_Type:_ `string`

The name of the new project

### --apiKey

_Type:_ `string`

The API key you'd like to generate the project using.
